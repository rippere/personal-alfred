# System Configs

## Overview
Documentation repository for Arch Linux system configurations and setup procedures.

## Purpose
Document system configuration for:
- Network setup and troubleshooting
- Window manager configurations
- Cloud service integrations
- System maintenance procedures

## System Information
- OS: Arch Linux 6.17.8-arch1-1
- Network: iwd v3.10 + systemd-networkd
- DNS: systemd-resolved
- Window Manager: Hypr

## Documented Configurations
- **Network**: eduroam and general network setup
- **Hypr**: Window manager configuration
- **Google Drive**: Cloud storage sync setup
- **System Procedures**: Restart and maintenance workflows

## Technologies
- Arch Linux
- iwd (wireless daemon)
- systemd-networkd
- systemd-resolved
- Hypr window manager

## Current Status
Living documentation repository. Updated as system configurations change or issues are resolved.

## Purpose in Ecosystem
Serves as system administration knowledge base and troubleshooting reference. Ensures repeatability of system setup and configuration.
